## `performance.now()`

https://developer.mozilla.org/en-US/docs/Web/API/Performance/now

```js

let t0 = performance.now();
doSomething();
let t1 = performance.now();
// console.log("Call to doSomething took " + (t1 - t0) + " milliseconds.");
console.log(`Call to doSomething took ${t1 - t0} milliseconds.`);

```
